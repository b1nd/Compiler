# Compiler/Translator
Pascal to C# compiler/translator.

Using ANTLR V4.7

Translating or/and compiling in C#

## Supported features

### Operators

+, -, *, div, mod, =, <>, >, <, >=, <=, and, or, not, &,Var, Begin, End

### Advanced

 Write(), Readln(), Writeln(), Procedure()
